* Overview

This repository contains a curated collection of code templates and boilerplate files to jumpstart development in various programming languages and environments. It includes both executable script templates and useful =.gitignore= files for different project types.

* Programming Language Templates

** Python Template
- Location: =python=
- Features:
  - Shebang for direct execution
  - PyQt5 process output reader class
  - ANSI to HTML conversion capabilities
  - Signal/slot architecture for GUI applications
  - Unicode text decoding

** Shell Script Template
- Location: =shell-script=
- Features:
  - Bash shebang with environment detection
  - ShellCheck integration
  - Sourced helper functions (=~/.qfuncs.sh=)
  - Error handling with =set -e -o pipefail=
  - Built-in help system
  - Command-line argument processing

** Swift Template
- Location: =swift=
- Features:
  - Swift shebang for script execution
  - Common import statements (commented)
  - Ready for Foundation, AppKit, and SwiftUI development

** Additional Templates
- =lua=: Lua script template
- =perl=: Perl/AppleScript template

* Documentation Templates

The =docs/= directory contains templates for the standard project documentation set. Each template includes structural guidance and placeholder text explaining what goes in each section.

See [[file:docs/README.md][docs/README.md]] for the full index and usage instructions.

** Included Templates
- =template-readme.md= — Project root README (overview, quickstart, docs index)
- =template-vision.md= — Purpose, problem, solution, design principles, non-goals
- =template-architecture.md= — System diagram, components, data flow, error handling
- =template-testing.md= — Test strategy, structure, IDs, Makefile targets
- =template-implementation-plan.md= — Phased delivery with issue tracking and roadmap
- =template-changelog.md= — User-facing release history (Keep a Changelog format)
